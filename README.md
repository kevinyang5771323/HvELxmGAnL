## 项目概述


## 技术栈

- 前端：Vue
- 后端：SpringBoot
- 数据库：MySQL 8
- 依赖管理：Maven
- Node.js版本：14
- Java版本：1.8

## 功能模块

### 前台模块

- 首页
- 宠物资讯
- 公告
- 宠物社区
- 宠物商品
- 宠物门诊
- 寄养中心
  - 支持挂号、寄养、点赞、评论等功能

### 后台模块

- 轮播图管理
- 公告栏管理
- 用户管理
- 食物管理
- 门诊管理
- 订单管理
- 社区管理
- 寄养管理

## 运行环境

- 后端：Java 1.8及以上版本，Maven构建
- 前端：Node.js 14及以上版本
- 数据库：MySQL 8

## 部署步骤

1. 准备开发环境，包括Java、MySQL、Node.js等。
2. 使用Maven构建后端工程。
3. 使用Node.js运行前端工程。
4. 初始化数据库，执行提供的SQL文件。
